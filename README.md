## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- Figma

## 💻 Projeto

O **DevLinks** é o meu primeiro grande passo no mundo do desenvolvimento web! 🚀 

Construí esse projeto do zero durante o programa **Discover da Rocketseat**, onde aprendi a criar um agregador de links totalmente responsivo. O diferencial aqui é a funcionalidade de **troca de tema (Light/Dark mode)**, que desenvolvi usando JavaScript para que o site se adapte à preferência do usuário com apenas um clique. 

Foi um desafio incrível colocar em prática conceitos de HTML semântico, estilização com CSS e a lógica de programação necessária para os efeitos visuais.
